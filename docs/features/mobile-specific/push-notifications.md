# Push Notifications

## Overview

Push notifications deliver real-time updates directly to users' mobile devices, keeping them informed about booking status, trip reminders, and important alerts without requiring them to actively check the app. Strategic use of push notifications enhances engagement, reduces missed pickups, and provides timely information when users need it most.
